## Getting the code
```
$ git clone https://github.com/amiralles/dsjssrc
$ cd dsjssrc
$ npm install
$ npm test
```

After running the commands from the section above, you should be ready to go. If something went bad, please don't hesitate to ask. I'll be happy to help!
